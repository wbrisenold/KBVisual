# Agent Memory System

CoALA-structured persistent memories for agentic tools working on KB Visualz.

## Structure

- `semantic/` — static facts about project architecture, design decisions, conventions
- `episodic/` — chronological records of sessions, changes, and outcomes
- `procedural/` — step-by-step workflows and operations (build, deploy, test)

## Usage

Read relevant memory files before starting new work. Append to episodic/ after each session.
